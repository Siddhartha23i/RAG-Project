SECTION 2: CHRONOLOGICAL TIMELINE (1975-2025)
================================================================================
[1975] Founded in Albuquerque to develop BASIC interpreters for the Altair 8800.
[1979] Headquarters moved to Bellevue, Washington.
[1980] Partnered with IBM to bundle an OS for the IBM PC. Microsoft licensed "86-DOS" to create MS-DOS.
[1985] Windows 1.0 released. First retail version of Windows.
[1986] IPO (Initial Public Offering) at $21/share.
[1990] Windows 3.0 launched; Office suite introduced (bundling Word, Excel, PowerPoint).
[1995] Windows 95 launched (Start button introduced). Internet Explorer bundled ("Browser Wars").
[1998] DOJ Antitrust Lawsuit filed (US v. Microsoft) regarding monopoly abuse.
[2000] Steve Ballmer becomes CEO. Bill Gates transitions to Chief Software Architect.
[2001] Windows XP released. Original Xbox console launched (entry into gaming hardware).
[2008] Azure (Cloud computing platform) announced; launched 2010.
[2009] Bing search engine launched.
[2011] Skype acquired ($8.5B).
[2012] Surface line launched (Microsoft's first major PC hardware). Yammer acquired.
[2013] Nokia mobile unit acquired ($7.2B) - later written off as a failure.
[2014] Satya Nadella named CEO. Shift to "Mobile First, Cloud First" strategy. Minecraft (Mojang) acquired ($2.5B).
[2016] LinkedIn acquired ($26.2B).
[2018] GitHub acquired ($7.5B).
[2019] Microsoft invests $1B in OpenAI. Teams surpasses Slack in daily users.
[2021] Market cap hits $2 Trillion. Windows 11 released. Nuance Communications acquired ($19.7B).
[2022] Activision Blizzard acquisition announced ($68.7B).
[2023] "New Bing" launched with GPT-4. Microsoft 365 Copilot announced.
[2023] Activision Blizzard deal closes (October) after regulatory battles with FTC/CMA.
[2024] Copilot key added to Windows keyboards (first layout change in decades).
[2025] "Project Stargate" (AI Supercomputer) plans leaked/rumored.
[2025] (Oct) New agreement with OpenAI: Microsoft holds ~27% stake in OpenAI Group PBC; IP rights extended to 2032.

SECTION 7: CONTROVERSIES & ANTITRUST
================================================================================
ANTITRUST (90s/00s):
- US v. Microsoft: Accused of monopolizing the PC market by bundling Internet Explorer with Windows to kill Netscape. Settled in 2001 (avoided breakup).
- EU Browser Choice Screen: Fined for failing to offer browser choice.

MODERN REGULATORY:
- Cloud Licensing: Accusations by EU providers (CISPE) regarding unfair licensing costs for running Windows/Office on non-Azure clouds (Settled 2024).
- Activision Blizzard: FTC and UK CMA blocked the deal initially; passed after concessions (cloud gaming rights sold to Ubisoft in certain regions).

SECURITY:
- SolarWinds Hack (2020): Microsoft code repositories accessed.
- Exchange Server Hafnium (2021): Massive zero-day exploits.
- Storm-0558 (2023): Chinese hackers stole signing key to access US gov emails.
